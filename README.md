# BlueBikes Analytics – Watertown Usage Dashboard

This project uses Bluebikes trip data to build interactive Tableau dashboards focused on Watertown and nearby stations, helping stakeholders understand usage patterns and support decisions about station planning.

## Objectives

- Visualize Bluebikes usage for Watertown stations and surrounding areas.
- Identify peak times, seasonal trends, and rider behavior.
- Compare Watertown station performance with nearby Boston/Cambridge locations.
- Provide clear visuals for presentations to Watertown and Bluebikes stakeholders.

## Data & Tools

- Bluebikes trip data (CSV files) with start/end stations and timestamps.
- Station/location data for mapping and station-level analysis.
- Tableau for data cleaning, calculations, and interactive dashboards.

## Key Dashboards

- **Trips Over Time:** Trips by hour, day of week, and month to reveal demand patterns.
- **Station Performance:** Trips starting/ending at each station, highlighting top and low-usage stations.
- **Origin–Destination Flows:** Movement between Watertown and neighboring zones.
- **Seasonality & Peaks:** Filters to explore specific months, seasons, or time windows.

## Results

- Identified high-demand stations and time periods in Watertown.
- Highlighted underused stations and potential opportunities for rebalancing or new locations.
- Dashboards were presented to Watertown stakeholders, who expressed interest in further collaboration.

## How to View

1. Download the `.twb` / `.twbx` Tableau workbook from this repository.
2. Open it in Tableau Desktop (or Tableau Public if applicable).
3. Use the filters and tooltips on each dashboard to explore Bluebikes usage in detail.
